## Publishing Steps

1. **Create Developer Account**
   - Go to: https://chrome.google.com/webstore/devconsole
   - Pay one-time $5 registration fee
   - Verify email

2. **Prepare Package**
   - Create ZIP of extension folder
   - Include all necessary files
   - Exclude: .git, node_modules, .DS_Store

3. **Upload Extension**
   - Click "New Item"
   - Upload ZIP file
   - Fill in store listing details

4. **Add Assets**
   - Upload icon (128x128)
   - Upload screenshots
   - Add promotional images (optional)

5. **Set Pricing & Distribution**
   - Free or paid
   - Select countries
   - Set visibility (public/unlisted)

6. **Submit for Review**
   - Review takes 1-3 days
   - Check for policy compliance
   - Wait for approval
